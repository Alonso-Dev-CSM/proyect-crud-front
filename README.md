# ProyectoCrud

No usar directamente AngularCLI

## Pasos
1. Clonar el .env.template y renombrarlo a .env
2. Llenar las variables de entorno
3. Crear Angular ENVS (opcional)

```
npm run envs
```

4. Para develoment ejecutar: 
```
npm run start
```

5. Para produccion ejecutar:
```
npm run build
```
